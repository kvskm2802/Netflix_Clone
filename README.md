## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/kvskmurty2802/Netflix_Clone.git
2. Navigate into the project directory:
   ```bash
   cd Netflix_Clone
3. Install dependencies:
   ```bash
   npm install
4. Start the development server:
   ```bash
   npm run dev
5. Open your browser and visit:
   ```bash
   http://localhost:5173
